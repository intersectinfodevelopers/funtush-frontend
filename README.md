# Funtush Frontend

Frontend application for the Funtush travel SaaS platform.

## Overview

Funtush is a travel SaaS solution designed to help users discover and book travel services with ease.

## Repository Contents
- `apps/web/` — main web frontend
- `apps/mobile/` — mobile app assets
- `apps/api/` — backend API definitions
- `packages/` — shared libraries and utilities
